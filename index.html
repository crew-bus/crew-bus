How It Works

Three commands. That's it.

Clone, install, run. Your crew is online in under a minute.

1
→

Clone the repo

Pull the code from GitHub. Everything is in one directory — no scattered config files.

2
→

Install PyYAML

One dependency. pip install pyyaml. No frameworks, no Docker, no cloud accounts.

3

Launch the dashboard

python dashboard.py — open localhost:8080 and your crew is live.

Quick Start

Up and running in 30 seconds

~/projects
$ git clone https://github.com/crew-bus/crew-bus.git
Cloning into 'crew-bus'... done.

$ cd crew-bus && pip install pyyaml
Successfully installed PyYAML-6.0.1

$ python dashboard.py
crew-bus Command Center starting...
■ Crew Boss (trust: 5) .... online
■ Guard ................... online
■ Wellness ................ online
■ Ideas ................... online
■ Wallet .................. online

Dashboard live at http://localhost:8080

Features

Everything you need. Nothing you don't.

Built for people who want AI that works for them — privately, locally, simply.

★

5 Core Agents

Crew Boss, Guard, Wellness, Ideas, and Wallet. Pre-configured and ready to coordinate.

⚙

Trust Score (1-10)

Set it to 1 and see everything. Set it to 10 and get a morning brief. You control the autonomy.

♥

Burnout Awareness

When you're running hot, non-urgent messages are held for better timing. Your crew respects your energy.

🔒

Private Sessions

Talk directly with any agent. Not even Crew Boss sees the content. Private means private.

☰

Teams & Hierarchy

Add departments with managers and workers. Scale from 5 agents to 50. Build your org chart.

🔔

Team Mailbox

Any agent can escalate directly to you if something critical is being ignored. No message gets silenced.

◉

Visual Dashboard

Clean circle layout with status dots. Mobile-first design. See your entire crew at a glance.

📄

Full Audit Trail

Every message, every decision, every routing event. Logged locally in SQLite. Nothing hidden.

⚡

Runs on Anything

Python + SQLite. Works on a Raspberry Pi, a laptop, or a server. No Docker. No cloud. Just run it.

Privacy

Privacy is real, not performative.

Everything stays on your machine. Period.

🔒

Truly private sessions

Crew Boss logs that a session happened but never sees the content. Your private conversations stay private.

📩

Sealed mailbox

Team mailbox logs that a message was sent but never the content. Metadata only.

🏠

Fully local

No cloud, no telemetry, no phone-home. Your agents run where you tell them to.

🗃

You own your data

It's a SQLite file on your machine. Back it up, delete it, move it — your choice.
