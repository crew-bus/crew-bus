FAQ

Is the core really free forever?

Yes. MIT License. The full message bus, Crew Boss, all 4 core agents (Guard, Wellness, Ideas, Wallet), dashboard, CLI, 3 free team templates (School, Passion Project, Household), and every feature listed under Core is free forever. No trial periods, no feature gates, no usage limits.

What does the $29 Guardian activation get me?

Guardian unlocks the Skill Store — downloadable skills that make any agent smarter. It also adds threat monitoring, anomaly detection, automatic escalation, and audit hardening. One-time $29 — no subscription. The activation key works forever.

Does one team license cover unlimited teams?

No. Each license covers one team. If you need 3 departments, that's 3 Department licenses. This keeps pricing fair — you only pay for what you use.

How many agents can I have per team?

Each team supports up to 10 agents — 1 manager + up to 9 workers. You can hire and terminate agents anytime from the dashboard.

What AI models are supported?

Six providers: Kimi K2.5 (free tier available), Claude, OpenAI, Groq, Gemini, and Ollama (fully local, no API key needed). You can pick a different model for each agent.

Is my data sent to the cloud?

No. crew-bus runs locally. SQLite database on your machine. No telemetry. No cloud accounts. The only network calls are to your chosen LLM provider and Stripe for payments.

⚡ Payments secured by Stripe. We never see your card details.
