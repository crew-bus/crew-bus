## What makes this different

Every AI tool today gives you a chatbot. You type, it responds, you move on.

Crew Bus gives you a **crew** ‚Äî a team of AI agents that know you, protect you, and work together behind the scenes so you can focus on living your life.

Your **Crew Boss** is your right hand. It handles 80% of everything ‚Äî messages, tasks, scheduling, coordination. It runs on the smartest model you can afford because it's worth it. - **Inner Circle Protocol** ‚Äî Your personal support agents talk exclusively to Crew Boss, never to you directly (unless you start a private 1-on-1 session). This protects your energy. You only hear what matters, when you're ready to hear it.

This isn't just an AI tool. It's the first AI system designed to genuinely protect the human using it.

## How It Works

```bash
git clone https://github.com/crew-bus/crew-bus.git
cd crew-bus
pip install pyyaml
python3 dashboard.py
```

Your browser opens. Guardian walks you through setup ‚Äî pick your AI model, paste your API key, and build your first crew from ready-made templates. No terminal knowledge needed after this.

Use `--no-browser` if you prefer headless mode.

### Example Crews

```bash

# Artist / passion crew ‚Äî for musicians, writers, painters, makers
```

Or skip the examples and let Guardian build your first team interactively.

## The Protection Stack

Three layers that no other AI system has:

1. **INTEGRITY.md** ‚Äî Injected into every agent prompt. Rules: never gaslight, never dismiss feelings, never sugarcoat, never dump bad news at bad times, validate first always. Real-time scanning catches violations. Heartbeat audits sweep every 30 minutes. Severity: high.

2. **CREW_CHARTER.md** ‚Äî Injected into every subordinate agent prompt. Rules: be honest, be competent, be caring, be disciplined, be efficient. No drama, no neediness, no manipulation. Two violations = fired. Crew Boss investigates reports and recommends firing to the human. Severity: medium.

3. **Guardian + Skill Vetting** ‚Äî Every skill that enters the system is scanned for prompt injection, data exfiltration, jailbreak attempts, and code execution. Dangerous skills are hard-blocked. Unknown skills need human approval. Only vetted builtins auto-approve. The vetting pipeline is behind the $29 Guardian activation.

4. **Skill Sandbox** ‚Äî After a skill is installed, Guardian never stops watching. Every agent response is timed and analyzed. Error rates, charter violations, integrity violations, and response time anomalies feed a real-time health score (0-100). Skills that degrade below safe thresholds get quarantined automatically ‚Äî removed from the agent, flagged in the audit log. Restore re-vets the skill before reinstalling. Zero tolerance for buggy or malicious behavior.

## Privacy

- **100% local.** Your data never leaves your machine. The only network calls are to your chosen LLM provider (for AI responses) and Stripe (for payments).
- **Private sessions are truly private.** Crew Boss logs that a session happened but never sees the content. Not even Crew Boss. Not even Guardian.
- **You own your data.** It's a SQLite file on your machine. Back it up, delete it, move it, read it with any SQLite browser ‚Äî your choice.
- **No telemetry. No tracking. No analytics. No ads.** This is your crew, on your hardware, running your rules.
